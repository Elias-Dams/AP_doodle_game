virtual void ISubject::Attach ( std::shared_ptr< IObserver >  observer)
pure virtual

Attach an observer to the subject.

This function attaches a pointer to an observer to the subject. This is a shared pointer because i the subject has full control over the deletion of the observer. e.g. if the platform disappears the view also gets destroyed.
